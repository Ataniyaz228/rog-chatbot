## Project Overview

Neural Core enables deep reasoning and analysis across your private document collections with a premium, high-tech interface.

### Key Features

*   **Intelligence:** Powered by the most advanced Claude Opus 4.6 model for superior reasoning.
*   **Deep Document Analysis:** Analyzes complex relationships across multiple documents with high precision.
*   **Advanced RAG Engine:** Efficient vector search and context management for relevant answers.
*   **Premium UI/UX:** A dark, futuristic interface designed with smooth animations, 3D elements, and professional typography.
*   **Smart Citations:** Automatic source tracking and section-specific references for transparency.
*   **Security First:** Environment-based configuration for sensitive API keys and data.

### Installation Guide

#### Prerequisites
*   Java 17 or higher
*   Node.js 18 or higher
*   PostgreSQL
*   Anthropic API Key

#### Backend Initialization

1.  Navigate to the backend directory:
    ```bash
    cd backend
    ```
2.  Create a `.env` file and configure your credentials:
    ```env
    AI_API_KEY=your_claude_api_key
    AI_MODEL=claude-opus-4-6-thinking
    AI_BASE_URL=https://api.anthropic.com/v1
    DB_PASSWORD=your_db_password
    ```
3.  Execute the application:
    ```bash
    ./mvnw spring-boot:run
    ```

#### Frontend Initialization

1.  Navigate to the frontend directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Обзор Проекта

Neural Core — это современная система генерации с дополненной выборкой (RAG), обеспечивающая глубокий анализ ваших личных коллекций документов через технологичный интерфейс премиум-класса.

### Основные Возможности

*   **Интеллект:** Использует самую мощную модель Claude Opus 4.6 для превосходного анализа.
*   **Глубокий Анализ:** Исследует сложные связи между несколькими документами с высокой точностью.
*   **Продвинутый RAG Движок:** Эффективный векторный поиск и управление контекстом.
*   **Премиальный UI/UX:** Темный, футуристичный интерфейс с плавными 3D-анимациями и строгой типографикой.
*   **Умное Цитирование:** Автоматическое отслеживание источников для полной прозрачности ответов.
*   **Безопасность:** Конфигурация через переменные окружения для защиты API-ключей.

### Руководство по Установке

#### Требования
*   Java 17 или выше
*   Node.js 18 или выше
*   PostgreSQL
*   Ключ API Anthropic (Claude Opus)

#### Настройка Бэкенда

1.  Перейдите в директорию бэкенда:
    ```bash
    cd backend
    ```
2.  Создайте файл `.env` и добавьте ваши настройки:
    ```env
    AI_API_KEY=ваш_ключ_claude
    AI_MODEL=claude-opus-4-6-thinking
    AI_BASE_URL=https://api.anthropic.com/v1
    DB_PASSWORD=пароль_бд
    ```
3.  Запустите приложение:
    ```bash
    ./mvnw spring-boot:run
    ```

#### Настройка Фронтенда

1.  Перейдите в директорию фронтенда:
    ```bash
    cd frontend
    ```
2.  Установите зависимости:
    ```bash
    npm install
    ```
3.  Запустите сервер для разработки:
    ```bash
    npm run dev
    ```
